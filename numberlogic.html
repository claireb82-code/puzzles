Skyline

Fill the 5×5 grid so every row and column has each height 1–5 exactly once. The numbers around the edge tell you how many buildings are visible from that direction — a taller building hides any shorter ones behind it.

How to read the clues

Each cell holds a building height from 1 (shortest) to 5 (tallest). No height repeats within a row or column.

A clue number is how many buildings you'd see looking down that row or column from that side — you can see a building only if every building before it (from your viewpoint) is shorter.

Click a cell to select it, then type 1–5 (or use the keypad). Backspace/Delete or the ✕ button clears a cell. A red outline means that value conflicts with another cell in its row or column.
